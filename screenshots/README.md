# Screenshots Directory for README.md

## Instrucciones para agregar capturas de pantalla:

1. **Registro de Usuarios** - `registro.png`
   - Mostrar el formulario de registro con campos de paciente/médico
   - Incluir validación de contraseñas

2. **Dashboard Principal** - `dashboard.png`
   - Vista principal después del login
   - Opciones de navegación clara

3. **Videollamada con Chat** - `videollamada.png`
   - Interfaz de videoconferencia activa
   - Chat interactivo visible

4. **Gestión de Citas** - `citas.png`
   - Formulario para crear nuevas citas
   - Lista de citas existentes

5. **Historial Médico** - `historial.png`
   - Vista del historial clínico
   - Detalles de consultas previas

## Cómo crear las capturas:

```bash
# Usar herramienta de captura de pantalla
# Recomendado: Snipping Tool (Windows) o Cmd+Shift+4 (Mac)
# Guardar en formato PNG con buen calidad
# Tamaño recomendado: 1200x800px
```

## GIFs opcionales:

- `videollamada.gif` - Demostración de videollamada en vivo
- `chat.gif` - Flujo de mensajes en tiempo real

```
screenshots/
```
